The NOC Thoughts Blog

The blog is maintained by me, Ken Perry, a Senior UC Engineer working for an MSP as part of a "Virtual NOC" (100% Remote). It is my hope that the how-to's, scripts, incident breakdowns and more that I blog about help other engineers in the field. Whether we're diagnosing failed Storage Adapters in a UCS-C CIMC, figuring out how to automate SQL query data pulls from a Cisco Call Manager, or virtual Lab setups, I'm sure there's something for you here.

If you're interested in my scripting work, check out my GitHub repo using the social icon at the top right of the site. I do want to state that I am a novice programmer and my primary language is Python (3.x). I offer no warranty and provide support for the scripts through GitHub. You can also email me. Some of the scripts I've created thus far are SQL Queries through AXL, Device registration state checks through RISPort and ASTS API, Port checking scripts, UC/UCC (UC VOS) health check scripts, and a few more things sprinkled in. I am actively developing scripts as I come across situations that warrant automation or the use of Python, so do check back for relevant blog posts.

For any questions, concerns, content requests or anything else you can email nocthoughts@gmail.com, and feel free to check out my socials.
